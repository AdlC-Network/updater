# Spigot updater

*This will go somewhere better when the docs get written properly*

## papermc (recommended)

Supports **major versions only** (`1.15`, `1.16` etc)

- `waterfall`
- `paper`
- `taverntine`

## serverjars

Supports minor versions (`1.16.1`, `1.16.2` etc) and `latest`.

- `nukkitx`
- `pocketmine`
- `magma`
- `mohist`
- `travertine`
- `bungeecord`
- `velocity`
- `waterfall`
- `bukkit`
- `paper`
- `spigot`
- `snapshot`
- `vanilla`
